# Airbnb Clone 🚀

Este é um projeto que replica as principais funcionalidades e o design do Airbnb, desenvolvido para consolidar conhecimentos em desenvolvimento web moderno.

## 🛠️ Tecnologias Utilizadas
- **Framework:** Next.js (App Router)
- **Biblioteca Visual:** React
- **Estilização:** Tailwind CSS
- **Linguagem:** JavaScript / TypeScript

## ⚙️ Como rodar o projeto localmente

1. Clone o repositório:
`git clone https://github.com/Luzia-Vitoria-Dev/Airbnb.git`

2. Instale as dependências:
`npm install`

3. Rode o servidor de desenvolvimento:
`npm run dev`
